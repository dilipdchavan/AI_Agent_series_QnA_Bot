
# Chat Bot Application - Streamlit
Here I have created, A simple chatbot application that answers user questions using predefined rules or AI-generated responses.
A conversational application that interacts with users and provides automated responses to their queries.
I used Streamlit to create an interactive web application for the chatbot.
Streamlit is an open-source Python framework for building interactive web applications for data science and AI with minimal code.
Streamlit enables developers to create interactive web apps quickly using Python.

--------------------
Need to import following files through

require2.txt
dotenv
langchain 
langchain-google-genai
langchain-openai
langchain-anthropic
streamlit
